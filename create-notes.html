Note Creation Workflow

A complete, step-by-step guide to generating and formatting Study Sphere notes using AI.

1. Doc Manager Pro

Screen Purpose: The starting point for creating, scanning, and formatting study notes.

🟪 Create in Docs

Tap to open Google Docs. You can create, edit, and collaborate on documents directly inside Google Docs.

🟩 Scan Doc

Tap to scan physical notes using your device camera. Scanned pages can be converted into digital documents and stored for future use.

🟥 Create Note

Tap to open the built-in StudySphere Note Editor. Designed for creating formatted study notes that can later be exported as TXT or PDF files.

ℹ️ Information Button

Tap the Information (i) button located in the top-right corner to open the User Manual and formatting guide.

3. ChatGPT Prompt Page

Screen Purpose: Configure ChatGPT to generate notes in the StudySphere format.

🟨 Steps to Follow:

  • 1. Open ChatGPT.
  • 2. Paste the copied StudySphere formatting prompt.
  • 3. Send the prompt.
  • 4. ChatGPT will now understand the required formatting structure.
  • 5. Ask your study question.

💡 Example Prompts:

"Thermodynamics 15 marks"
or
"Art and Architecture of Medieval India"

ChatGPT will generate content following the StudySphere formatting rules.

4. Generated Format

Screen Purpose: ChatGPT generates notes that are already formatted for StudySphere.

🟩 What You Will Receive

The output may contain: Headings, Definitions, Keywords, Facts, Timelines, Tables, Flow Charts, Examples, Revision Blocks, Exam Tips, Memory Tricks, and Summaries.

⚡ Ready to Use

The generated response is already structured and can be copied directly into the StudySphere Note Editor without additional formatting.

5. Create Note

Screen Purpose: Open the StudySphere Note Editor to begin assembly.

🟥 Create Note Button

Tap Create Note to launch the editor. This is the main workspace used for creating notes. Here you can paste your ChatGPT notes, edit content, add headings, format text, preview, and export.

6. Note Editor

Screen Purpose: Create, edit, and export your study notes.

🟩 Note Title

Enter the title of your note (e.g., Thermodynamics).

🟦 Quick Formatting Toolbar

Use these to structure notes without manually typing symbols:
# Main Heading | ## Sub Heading | ### Section Heading
* Bold Text | = Highlight Text

🟨 AI Assistant

Opens StudySphere AI features for note assistance and enhancement.

🟧 Export Menu

Opens export options for your finalized notes.

🟪 Export as TXT

Save as a plain text document. Best for editing later, sharing text, or backups.

🟥 Export as PDF

Convert into a formatted PDF document. Best for printing, exam prep, offline reading, or sharing.
